========================================
  全文検索システム - 社内配布用 インストール手順
========================================

【配布の形態】
  ・ MSIX で配布された場合 … 証明書（.cer）と .msix が同梱されています。
  ・ ZIP で配布された場合 … 解凍して exe を実行するだけです（証明書不要）。

  共通で必要なもの: .NET 8 デスクトップ ランタイム（未導入の場合のみ）
  Python のインストールは不要です。

【動作環境】
  - Windows 10/11（64bit）
  - .NET 8 デスクトップ ランタイム（未導入の場合は Microsoft のサイトからインストール）
  - （任意）Microsoft Office … Word/Excel 等のプレビュー表示用

----------------------------------------
  ■ MSIX で配布された場合のインストール
----------------------------------------

  1. 証明書のインストール（初回のみ）
     配布された「証明書ファイル（.cer）」をダブルクリックし、
     「証明書のインストール」→「信頼されたルート証明機関」に
     インストールしてください。（ローカル コンピューターを選択）

  2. アプリ（MSIX）のインストール
     配布された「.msix」ファイルをダブルクリックし、
     表示される手順に従ってインストールしてください。
     .NET ランタイムが入っていない場合は、案内に従ってインストールしてください。
     インストール後、スタートメニューから「全文検索システム」を起動できます。

----------------------------------------
  ■ ZIP で配布された場合のインストール
----------------------------------------

  1. 配布された ZIP を解凍します。
  2. 解凍したフォルダ内の「FileSearch.Blazor.exe」をダブルクリックして起動します。
     証明書のインストールは不要です。
     （ウィンドウのタイトルバーに「全文検索システム」と表示されます。）

----------------------------------------
  ■ 初回起動後の設定（MSIX/ZIP 共通）
----------------------------------------

  1. 画面右上の設定ボタンをクリック
  2. 「検索対象フォルダ」に検索したいフォルダ（例: 社内ファイルサーバのパス）を追加
  3. 「保存」をクリック
  4. メイン画面で「インデックスを再構築」をクリックし、インデックス作成が終わるまで待つ
  5. 検索ボックスにキーワードを入力して Enter で検索。左のツリーでファイルをクリックすると右にプレビューが表示されます。

【トラブルシューティング】
  - MSIX がインストールできない／「不明な発行元」と出る
    → 上記「証明書のインストール」を先に実施してください。（MSIX 配布時のみ）
  - アプリが起動しない
    → .NET 8 デスクトップ ランタイムがインストールされているか確認してください。
      https://dotnet.microsoft.com/download/dotnet/8.0 の「ランタイム」からインストールできます。
  - 不明な点は配布元（社内）にお問い合わせください。
